# json输出格式
## `{uid}.json`
* 格式
```json
[
    [
        "group_id",
        "id",
        "title",
        "state",
        "current_progress",
        "target_progress",
        "amount",
        "date"
    ],
    [
        "成就group_id",
        "成就id",
        "成就标题",
        "完成状态",
        "当前进度",
        "目标进度",
        "完成次数",
        "完成日期"
    ],
]
```
